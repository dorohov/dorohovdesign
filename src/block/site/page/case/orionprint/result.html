Клиенты стали приходить из тех каналов, которые раньше не работали: из поисковиков, заявки из контекстной рекламы, обращения из соцсетей, звонки (в день в типографию звонят около 30 раз).

Из 52 заявок — 35 превратились в оформленные заказы.

Из соц. сетей в месяц приходит в среднем около 28 обращений.

Поисковой трафик увеличился с 0 до 1200 посетителей в месяц

Основными источниками захода на сайт стали: органическая выдача, переходы по рекламе.
